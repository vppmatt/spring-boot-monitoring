Payments - sample application

This application is a sample Spring Boot Rest application. It contains a list of payments made for orders for a service such as an insurnace policy or an entertainment subscription. Each order can have multiple payments (e.g. some orders are paid for monthly, others as single payments).

Instructions

Get requests can be issued as follows:

http://localhost:8080/api/payments - retrieves the first 100 records

http://localhost:8080/api/payments?country=swe - retrieves all records for the specified country (warning could be slow if there is a lot of data!)

http://localhost:8080/api/payments?order=21213379 - retrieves all records for the specified order

http://localhost:8080/api/payments/82 - retrieves the specific record

http://localhost:8080/api/country - retrieves a list of all countries in the system
